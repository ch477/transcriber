Transcriber-1.4.4 download page

Source distribution

  • sources of Transcriber 1.4.4 (~ 1 Mb gzipped archive) released on 04/19/02.
You will also need:
  • Snack (2.0.2 or higher) for multi-platform audio driver - and much more other useful things! A source distribution of Snack 2.1 is mirrored locally.
  • tcLex1.2 a flex-like parsing extension for Tcl,
  • Tcl/Tk version 8.0 or later for the interface and more... (if you do not already have it).
And optionaly, as an extension for the Snack module:
  • Sphere_2.6a for specific NIST/SPHERE audio format access.

Binary distributions

Binary distributions are not yet available for all platforms with the latest version of the tool, so check the version! If you do not find a binary for your platform, either build it from the sources (above) and please make it available to others, or download a binary for an older version Transcriber 1.4.3 (for Linux/Macintosh), or Transcriber 1.4.2. For ease of use, Snack and tcLex modules are included in most distributions, but Tcl/Tk has to be installed separately. Be careful to install only a distribution matching exactly your Tcl/Tk version.
  • for Linux/x86
    • Debian package for use with Debian GNU/Linux compatible distributions (~ 0.5 Mb package, requiring Snack (>=1.6.5) and TcLex). With many thanks to David A. van Leeuwen. This package is mirrored locally (Transcriber 1.4.4-1).

Sound sample

A 20 seconds sample with its transcription is included in the source distribution. You can test the tool on a longer example with:
  • a 150 seconds sample of French radio in 8kHz mu-law wav format (1.2 Mb) and
  • the transcription in Transcriber's native XML format (<7 kb)
