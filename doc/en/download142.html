Transcriber-1.4.2 download page

Source distribution

  • sources of Transcriber 1.4.2 (~ 940 kb gzipped archive) updated on 8/2/00.
You will also need:
  • Snack 1.6 for multi-platform audio driver - and much more other useful things!
  • tcLex1.2 a flex-like parsing extension for Tcl,
  • Tcl/Tk version 8.0 or later for the interface and more... (if you do not already have it).
And optionaly, as an extension for the Snack module:
  • Sphere_2.6a for specific NIST/SPHERE audio format access.

Binary distributions

Binary distributions are not always synchronized for all platforms with the latest version of the tool, so check the version! For ease of use, Snack and tcLex modules are included in most distributions, but Tcl/Tk has to be installed separately. Be careful to install only a distribution matching exactly your Tcl/Tk version.
  • for Windows
    • TclPro-wrapped binary distribution 1.4.2 including Tcl/Tk 8.2 and Snack 1.6 (~ 3.8 Mb self-installing executable including tcLex 1.2a1). For Windows 95/98/NT/2000. For installation, execute the file (by double clicking on it) and folow the installation instructions.
  • for Linux/x86
    • Debian package for use with Debian GNU/Linux compatible distributions (~ 0.5 Mb package, requiring Snack (>=1.6.5) and TcLex). With many thanks to David A. van Leeuwen.
    • RPM package for use with Mandrake 7.0 / 7.1 or Redhat 6.1 / 6.2 GNU/Linux distributions (~ 0.9 Mb package, requiring Snack 1.7 and TcLex 1.2). With many thanks to Damien Mercier. The 3 packages are also mirrored locally (Transcriber, Snack, and TcLex).
    • Binary distribution 1.4.2 for Tcl/Tk 8.2 or higher using 'stubs' mechanism'  (~ 1.2 Mb gzipped archive, including Snack 1.6.5 and tcLex 1.2a1, compiled on Linux Debian 2.0 - libc6 with Tcl/Tk 8.2.1)
    • Binary distribution 1.4.2 for Tcl/Tk 8.0.5 (~ 1.2 Mb gzipped archive, including Snack 1.6.5 and tcLex 1.2a1)
  • for Solaris
    • Binary distribution 1.4.2 for Tcl/Tk 8.0.5 (~ 1.2 Mb gzipped archive, compiled on Ultra-2 with Solaris/SunOS 5.5 and Tcl/Tk8.0.5, including Snack1.6.5 and tcLex 1.2a1)
  • for Silicon Graphics
    • Binary distribution 1.4.2 for Tcl/Tk 8.0.5 (~ 1.4 Mb gzipped archive, compiled on IRIX 6.2 with Tcl/Tk8.0.5, including Snack1.6.5 and tcLex 1.2a1).

Sound sample

A 20 seconds sample with its transcription is included in the source distribution. You can test the tool on a longer example with:
  • a 150 seconds sample of French radio in 8kHz mu-law wav format (1.2 Mb) and
  • the transcription in Transcriber's native XML format (<7 kb)
